เงื่อนไขประกอบการสั่งจ้าง
1. ผู้ขายต้องส่งมอบงานซึ่งปรากฏอยู่ในใบสั่งจ้างนี้ พร้อมด้วยหนังสือส่งมอบงาน หรือสำเนาใบส่งของ {{ data.delivery_doc_count }} ฉบับ
2. กำหนดส่งมอบงาน ภายใน {{ data.delivery_days }} วัน โดยนับถัดจากวันที่ได้รับใบสั่งจ้าง
3. ครบกำหนดส่งมอบงาน วันที่ {{ data.delivery_due_date }}
4. สถานที่ส่งมอบงาน {{ data.delivery_place }}
5. ระยะเวลารับประกันคุณภาพ {{ data.warranty_period }} ปี
6. การไฟฟ้าส่วนภูมิภาค สงวนสิทธิ์ค่าปรับกรณีส่งมอบงานเกินกำหนดเวลา ซึ่งผู้รับจ้างต้องถูกปรับตามข้อ 6.1
6.1 ถูกปรับเป็นรายวันในอัตราร้อยละ {{ data.penalty_rate_daily }} ของงานจ้างรวมภาษีมูลค่าเพิ่มที่ยังไม่ได้รับมอบ
6.2 กรณีต้องการผลสำเร็จของงานทั้งหมดพร้อมกัน ถูกปรับเป็นเงินรวมภาษีมูลค่าเพิ่มวันละ {{ data.penalty_lump_sum }} บาท (ในอัตราร้อยละ {{ data.penalty_rate_lump }} ของราคางานจ้างรวมภาษีมูลค่าเพิ่ม แต่ต้องไม่ต่ำกว่าวันละ {{ data.penalty_min_daily }} บาท)
7. การไฟฟ้าส่วนภูมิภาค สงวนสิทธิ์ที่จะไม่รับมอบงานนั้น ถ้าปรากฏว่างานนั้นมีลักษณะไม่ตรงตามรายการที่ระบุไว้ในใบสั่งจ้าง กรณีนี้ผู้รับจ้างต้องดำเนินการแก้ไขให้ถูกต้องตามใบสั่งจ้างทุกประการ ด้วยค่าใช้จ่ายของผู้รับจ้างเอง และระยะเวลาที่เสียไปเพราะเหตุดังกล่าว ผู้รับจ้างจะนำมาอ้างเป็นเหตุขอขยายเวลาทำการตามใบสั่งจ้าง หรือของดหรือลดค่าปรับไม่ได้
8. การประเมินผลการปฏิบัติงานของผู้ประกอบการ การไฟฟ้าส่วนภูมิภาคสามารถนำผลการปฏิบัติงานแล้วเสร็จตามสัญญา หรือข้อตกลงของคู่สัญญา เพื่อนำมาประเมินผลการปฏิบัติงานของผู้ปฏิบัติงาน
